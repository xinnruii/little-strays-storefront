# Little Strays

A boutique pet goods storefront built with Next.js, TypeScript, and modern e-commerce architecture.

Live site: [littlestrays.co](https://littlestrays.co/)

> Development began Sep 2025. Site launched Nov 2025. Repository migrated to GitHub Apr 2026.

## Tech Stack

Next.js · TypeScript · React · Supabase / PostgreSQL · BigQuery · gRPC · REST APIs · Vercel · Docker · Jest

## Architecture

Little Strays is structured as a modular e-commerce MVP:

* `little-strays-storefront` — customer-facing Next.js storefront (this repo)
* `little-strays-api` — backend API service (private)
* `little-strays-genai-service` — AI-powered product discovery and content generation (private)
* `little-strays-data` — analytics, event tracking, and data modeling (private)

## Highlights

* 32 SKUs across 5 product categories with structured REST/gRPC endpoints serving catalog, orders, and customer data
* GenAI recommendation engine analyzing purchase history and behavioral signals — conversion rate lifted from 15% to 23%
* Market intelligence pipeline informing sourcing decisions — 26% of revenue from newly introduced product lines
* BigQuery analytics infrastructure tracking traffic, behavior funnels, and conversion trends
* SEO and Core Web Vitals optimizations — 2× organic traffic growth over 6 months
* 81% test coverage, CI/CD on Vercel with automated preview deployments
