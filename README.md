# little-strays
A boutique pet goods storefront built with Next.js, TypeScript, and modern e-commerce architecture.

## Architecture

Little Strays is structured as a modular e-commerce MVP:

- `little-strays-storefront`: customer-facing Next.js storefront
- `little-strays-api`: backend API service
- `little-strays-genai-service`: AI-powered product discovery and content generation
- `little-strays-data`: analytics, event tracking, and data modeling
